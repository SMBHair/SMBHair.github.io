Format
This year’s tournament will be a Division C Science Olympiad Invitational tournament held on February 18, 2023, in-person on the UMass Amherst campus.
Our tournament will run all 23 main events this year, following all event rules according to the Science Olympiad Division C Rules Manual for 2023 unless noted otherwise. Our events will run with a different schedule than the national tournament; however, all event conflicts described in the 2023 manual will stay consistent in our tournament. We expect to provide individual homerooms for all competing schools.
This year, our tournament will focus on being hands-on. This means two big things for competing teams:
  1. The tournament will have a focus on build events, and
  2. Most1 non-build events will be run in a stations format.
